Freckler is an open source cognitive character technology platform.

For a technical introduction, please see the architecture + dependencies presentation on the experimental Friendularity.org site.

Our software is built using:

  • Java, Scala, and OSGi core technology
  • Basic symbolic smarts from Appdapter semantic computing platform
  • 3D character rendering to OpenGL using JMonkey Game Engine
  • Integration with MIDI + SMPTE music+video technology for action mapping, triggering, and synchronization
  • Serialization to Avro, cloud integration via Hadoop
  • Connection with many other popular open source mathematics, AI and robotics tools

For more information, please visit the wiki in our workspace.

You may ask questions and discuss the software using the cogchar-users google group.
